Soft IT – React (Vite) Template Documentation

Overview

This is a modern and responsive business landing page built with React (Vite) and Tailwind CSS. It is suitable for IT companies, startups, portfolios, and landing pages.

Folder Structure

📁 Template/
├── 📁 dist/                # Build files
├── 📁 public/              # Static assets
├── 📁 src/                 # Source code
│   ├── assets/
│   ├── components/
│   ├── pages/
│   └── App.jsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.js

Installation

To run this template locally, follow these steps:

  1. Install Node.js from nodejs.org
  2. Open terminal in the project root
  3. Run the following commands:

npm install
npm run dev

Build for Production


npm run build

Customization

All components are located in the src/components directory. You can easily update texts, images, and styles from there.
